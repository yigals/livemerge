A live real-time Turing test, inspired by
http://graphics.stanford.edu/~niloy/research/emergence/emergence_image_siga_09.html.

Our brain can identify the moving objects by a process called "Emergence" (en.wikipedia.org/wiki/Emergenceý), yet a computer cannot.

Output example:
http://www.youtube.com/watch?v=Whb09RyOOro

Installation:
Download Python, preferably 2.7.6 - http://www.python.org/getit/
Install OpenCV's python module from http://www.lfd.uci.edu/~gohlke/pythonlibs/#opencv

How to run:
open a command prompt, and run: python livemerge.py
